# TaskFlow Pro 🚀

Ye ek professional Task Manager app hai jo maine as a Developer apne portfolio ke liye banayi hai.

### ✨ Features:
- **Aesthetic Design:** Vibrant Pink aur Glassmorphism look.
- **Score System:** Har task poora karne par points milte hain.
- **Confetti:** Task khatam hone par celebration animation hoti hai.
- **Docker:** Ye app Docker par asani se chal sakti hai.

### 🛠️ Tech Used:
- HTML, CSS, JavaScript
- Docker & Nginx
